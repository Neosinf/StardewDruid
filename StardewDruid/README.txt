STARDEW DRUID

Requires SMAPI version 3.18.5 +

Optimised for:
- PC
- Single Player
- Vanilla Content

Key mechanic binded to MouseX1 and MouseX2
- can be changed in mod configuration after installing

Custom data model stored in save files
- only affects the mod, so won't impact any save files after uninstalling/installing

Seems to be compatible with most popular mods, including
- SVE
- Content Patcher
- JSON Assets

Recommended download via Nexus Vortex Mod Manager application
